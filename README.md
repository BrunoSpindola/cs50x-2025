# CS50x 2025 — Harvard University

Problem sets completed as part of Harvard's CS50x Introduction to Computer Science.

## Week 1 — C

| Project | Description |
|---|---|
| mario.c | Builds a pyramid of hashes using loops and conditions |
| cash.c | Calculates minimum coins for change using a greedy algorithm |
| credit.c | Validates credit card numbers using Luhn's algorithm |

## Week 2 — Arrays & Cryptography

| Project | Description |
|---|---|
| caesar.c | Encrypts messages using Caesar's cipher |
| substitution.c | Encrypts messages using a substitution cipher |
| readability.c | Calculates the reading level of a text using Coleman-Liau index |
| scrabble.c | Determines the winner of a Scrabble round by computing word scores |

## Week 3 — Algorithms

| Project | Description |
|---|---|
| plurality.c | Simulates a plurality voting election |
| runoff.c | Simulates a ranked-choice runoff election |

## Technologies
- C (Weeks 1–3)
- Python, SQL (upcoming weeks)
